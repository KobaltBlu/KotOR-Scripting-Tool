How to find


The find to can be used for finding like entries in your script.

To open the find window you will either have to select the Find option from the Edit tab on the Menubar located at the top of the application, or you can use the shortcut key Ctrl + F.

To find enter your query in the "Find What:" textbox and click Find Next.

Note* You can also check Match Case if you would like a case specific search.
