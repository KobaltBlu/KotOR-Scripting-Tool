Keyboard Shortcuts

I have tried to include as many helpful keyboard shortcuts that I can think of that will help during the scripting process so the user can spend more time in thier scripts and less time in the menus

Below you will find a list of all docuemnted shortcuts

Ctrl + N New file
Ctrl + O Open file
Ctrl + S Save file
Ctrl + Shift + S Save file as
Ctrl + Shift + C Compile file
Ctrl + F Find in file
Ctrl + H Find and Replace text in file
